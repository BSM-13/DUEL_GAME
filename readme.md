Python implemented turn based duel game where two characters with special abilities fight.

Features:

	1. The Duel - two randomly generated characters fight
	2. Statistics Test - The user decides the two characters' specifications and the program runs 1000 simulations and prints the results. The user can choose if he wants the program to print the battles in a .txt file, the console or not at all, as to make the experience cleaner.
	3. To add another ability, the user can write the code for it in the abilities file, add it to the list in the characters class list and to the main class list (sometimes depends on the ability you want to add).
	4. The program accepts the "--per-round" command line flag so the characters take new special abilities at the end of every round.
